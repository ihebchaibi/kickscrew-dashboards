## Answers To Amy's Questions

### 1. Should we separate languages into different campaigns?

Recommendation: Separate by market first, then test language inside each market before splitting fully by language.

Rationale:

- KSA should get Arabic-first creative and copy testing because it is the largest ME market and likely benefits most from localized messaging.
- UAE can run mixed English/Arabic tests because the audience is more multilingual.
- Do not split every campaign by language immediately, because that may fragment learning and budget too much.
- If Arabic and English show clearly different CTR/CPO after enough spend, then split into dedicated language campaigns or ad groups.

Suggested action:

- For Meta/TikTok: run Arabic and English creative variants inside KSA and UAE campaigns first.
- For Google Search: separate Arabic and English keyword groups/campaigns if search volume supports it.
- For PMax: use asset groups or market-specific assets rather than over-splitting too early.

### 2. Should we separate Brand and Generic keywords by core SKU?

Recommendation: Yes for Search. Brand and Generic should be separated, and high-volume core SKUs deserve their own controlled groups when volume is meaningful.

Rationale:

- Brand keywords usually carry higher intent and should not be blended with generic sneaker searches.
- Generic keywords need tighter budget control, negatives, and SKU-level learning.
- Core SKUs/categories such as Jordan, Nike, Adidas, Samba, AF1, Pokemon, or high-demand drops should be separated when they have enough volume.

Suggested structure:

- Brand Search: KICKS CREW terms and brand-protection queries.
- Generic Category Search: sneakers, shoes, basketball shoes, streetwear, etc.
- Core SKU / Drop Search: only for products with proven demand or active promotions.
- Competitor / marketplace terms: optional, low-budget test only.
