# KICKS CREW ME Command Center

Use this folder as the working memory for KICKS CREW ME strategy and execution.

## Core Notes

- `Active_Campaigns.md` tracks campaigns, markets, channel status, and next actions.
- `Decision_Log.md` captures decisions with dates, reasons, and follow-ups.
- `Creative_Learnings.md` stores ad creative learnings by channel and market.
- `Meta_Testing_Framework.md` defines how Meta tests should be launched, judged, scaled, paused, or killed.
- `Report_Format_Standard.md` defines the shared Slack Canvas style for daily, weekly, and monthly reporting.
- `Live_Data_Aggregation_Spec.md` defines how Windsor.ai pulls should be grouped, cleaned, and calculated.
- `Daily_Report_Template.md`, `Weekly_Report_Template.md`, and `Monthly_Report_Template.md` are the reusable reporting templates.
- `Live_Data_Setup.md` tracks connected Windsor.ai sources.

## Rule

Deliverables live in the main H workspace. Reusable learnings and decisions live here, then mirror to Obsidian when useful.

Reports should follow the same Slack Canvas style every time: verdict, #1 issue, at-a-glance table, channel breakdown, Shopify ground truth, GA4 funnel, revenue cross-check, then top 3 actions.
